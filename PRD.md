# empEnglish 首版 PRD 高层框架（V1.0）

> 目标：基于既有调研与方案，输出一份**可以直接指导团队对齐与启动开发**的首版 PRD 高层框架，而非细节实现文档。

---

## 一、产品概述

### 1.1 产品定义

**产品名称：** empEnglish  
**产品类型：** 考研复试英语口语 AI 智能体（微信小程序 + Web）  
**一句话定位：**  
> 面向西安高校考研生的「院校 / 专业定制型复试英语口语 AI 导师」，让你在正式复试前，先在手机上经历 10 场“本校本专业”的高还原模拟面试。

### 1.2 产品目标

- 帮助考研生在短时间（15–30 天）内，有针对性地提升**复试英语口语表现**；
- 专注西安高校起步，切出“**西安本地百万级市场 + 全国千万级细分盘**”；
- 通过差异化（本地化 + 专业化）在头部全国性口语 APP（可栗口语、咕噜口语等）旁边找到稳固位置。

### 1.3 商业目标（首年）

- 在西安本地实现 **\$20万+ 年经常性收入（ARR）**；
- 累积 **2,000+ 月活用户**，免费转付费转化率 ≥ 5%；
- 打通 3–5 家本地考研机构的 B 端合作试点。

## 二、市场与竞品背景（高层）

### 2.1 市场机会

- 全国考研报名人数常年在 380–430 万区间，英语复试口语是高频痛点[1]；
- 针对复试口语的 AI 陪练产品单款用户规模可达数百万级[3][4]；
- 西安高校密集（西交、西工大、西电、西外、西北大等），本地考研人数 10–15 万，保守估算可支撑 **\$100–\$300 万/年** 的本地复试口语服务空间。

### 2.2 竞品要点（可栗口语、咕噜口语、星空外语等）

- 已跑通的主流功能组合：
  - AI 导师面试系统（多轮模拟、突发追问）[1][2][3][4]；
  - 音素级发音纠错与多维度评分[1][9]；
  - 压力渐进式训练模式；
  - 学术表达增强与学习报告[4][5][13]。
- 通用定价带：
  - 冲刺/专项包：约 \¥98–\¥199；
  - 年度会员：约 \¥298–\¥398[1][16][17][19][20]。

### 2.3 差距与机会

- 竞品优势：技术成熟、功能全面、全国覆盖；
- 明显空白：
  - 「**针对某一城市 / 某一批高校 / 某一专业群的深度本地化**」几乎为空白；
  - 其“院校定制”更多是轻度题库调整，**未真正还原本校导师风格与专业语境**。

**empEnglish 的机会：**  
不和全国竞品拼“谁更泛用”，而是把「**西交/西工大/西电/西外** 的考研复试口语」做到最懂、最好用。

## 三、目标用户与使用场景

### 3.1 目标用户

- **主目标人群：**
  - 西安及周边高校考研学生（含一战/二战、自考跨考）；
  - 目标院校优先：西交大、西工大、西电、西外、西北大、西财、西邮等。
- **典型特征：**
  - 初试成绩刚出，距离复试 1–2 个月；
  - 对英语口语紧张，自评“开不了口 / 一紧张就卡壳”；
  - 已经或准备为考研辅导付费，对复试口语有强烈焦虑感。

### 3.2 使用场景

1. **基础适应期（复试前 1–2 个月）**
   - 目的：摸清自己的口语水平，了解复试问什么、怎么问；
   - 行为：每天 1–2 场通用模拟（自我介绍、家庭、专业背景等），熟悉流程。

2. **冲刺强化期（复试前 2–4 周）**
   - 目的：紧密围绕“目标院校 + 目标专业”提高匹配度；
   - 行为：购买冲刺包，使用院校定制题库 + 高压模式，完成 10–20 场“本校本专业”模拟。

3. **临考模拟期（复试前 1 周）**
   - 目的：真刀真枪适应“被盘问感”和心理压力；
   - 行为：每天 1 场 15–20 分钟「整场模拟」，查看报告，最后查缺补漏。

## 四、产品定位与价值主张

### 4.1 产品定位

- **垂直场景：** 只做「考研复试英语口语」，首发只服务西安；
- **差异化：**
  1. **本地高校 / 专业深度定制**：校–院–专业三级题库 + 导师风格包；
  2. **短周期高强度冲刺**：15–30 天高频练习，而非全年慢学；
  3. **线下资源结合**：联合本地机构、学长学姐，形成“AI 工具 + 人”闭环。

### 4.2 面向用户的核心卖点（对外版本）

1. **更对口**：  
   - 「西交电气 / 西工大航宇 / 西电通信 / 西外翻硕」等**专业专用题库**；
2. **更像复试**：  
   - 模拟“你真正会遇到的导师”——本校导师风格 + 专业追问；
3. **更省时间**：  
   - 用 15–30 天，刷够 10–20 场高还原模拟，所有弱点一目了然。

## 五、功能规划（高层架构）

> 采用「三层结构」：基础功能层 → 进阶功能层 → 本地化差异化层。  
> 每一层都标注：是否为 MVP 必须、以及其免费/收费策略。

### 5.1 基础功能层（MVP 必须，部分免费）

#### 5.1.1 语音交互基础能力

- **功能点：**
  - 语音输入，自动转写为英文文本（ASR）；
  - AI 导师语音播报问题与反馈（TTS）。
- **用户体验要求：**
  - 录完 1–2 秒内显示文本与评分；
  - 移动端环境下稳定不卡顿。

#### 5.1.2 通用复试流程模拟

- **功能点：**
  - 标准流程：自我介绍 → 本科背景 → 论文 → 实习 → 报考动机 → 职业规划 → 自由提问；
  - 一套「全国通用」的基础问答场景。
- **MVP 范围：**
  - 10–20 个通用话题，每话题 3–5 个典型问题；
  - 支持一键“开始一场通用模拟复试”。

#### 5.1.3 基础评分与纠错

- **评分维度（对齐官方标准）[10][11][12]：**
  1. 发音
  2. 词汇多样性
  3. 语法准确性
  4. 流利度与连贯性
- **展示：**
  - 单题评分 + 场次综合评分；
  - 每题 1–2 条简短建议。

#### 5.1.4 练习记录与基础错题本

- **功能点：**
  - 保存最近 3 天的通用练习记录（免费用户）；
  - 支持查看原回答文本 + 系统评分和建议；
  - 用户可标“需重练”并快速重做。

**免费策略：**  
通用流程 + 基础评分 + 3 天记录为永久免费，用于拉新和建立信任。

---

### 5.2 进阶功能层（竞品对齐，收费为主）

#### 5.2.1 压力渐进式训练模式

- **模式：**
  1. 温和鼓励型：更多提示与鼓励；
  2. 正常严谨型：标准复试语气；
  3. 高压盘问型：更频繁追问与质疑。
- **行为差异：**
  - 语气、追问频率、容忍停顿时间不同；
  - 高压模式会在用户明显卡壳时提前提出新问题，增强压迫感。

#### 5.2.2 突发追问引擎

- **逻辑：**
  - 基于回答中的关键词（公司名、课题、技术名词等）生成针对性追问；
- **目标：**
  - 逼近真实导师“顺着你说的再问一层”的体验。

#### 5.2.3 发音诊断与可视化

- **能力：**
  - 逐词发音评估：标记错误或明显中式发音；
  - 句子层面节奏与重音反馈。
- **可视化：**
  - 对应文本中，将问题单词高亮，并给简短提示；
  - 简化“热力图”展示整段发音问题集中位置。

#### 5.2.4 学术表达增强

- **功能：**
  - 在纠错之后，再输出一版「更学术、更高级」的优化回答；
  - 显式标出使用的高级表达与句型，供用户模仿。

#### 5.2.5 学习报告

- **报告类型：**
  - 每日小结：当天场次的平均分 + 维度雷达图 + 建议；
  - 周度报告：一周趋势图 + 高频错误列表 + 推荐训练计划。
- **导出：**
  - 付费用户可导出 PDF（用于发给老师/学长点评）。

**收费策略：**  
进阶功能整体作为付费包核心内容，仅提供少量体验场次给免费用户试用。

---

### 5.3 本地化差异化功能层（核心护城河，付费主力）

#### 5.3.1 西安高校 / 专业定制题库（核心差异化）

- **结构：**
  - 学校级（西交、西工大、西电、西外、西北大、西财、西邮……）；
  - 学院级（电气学院、航天学院、通信工程学院、高翻学院等）；
  - 专业级（电气工程、航空宇航、信息与通信、翻译硕士、金融学等）。
- **内容：**
  - 每专业：10–30 个高频问题（结合真题与经验贴）；
  - 特别包含：
    - 跨考/二战/非全日制等特殊身份的典型提问；
    - 本专业核心课程与研究方向相关问题。

#### 5.3.2 导师风格数字孪生

- **风格标签：**
  - 学术深挖型 / 实践导向型 / 友好交流型 / 刻板高压型；
- **表现：**
  - 提问角度不同：偏理论、偏实践、偏价值观等；
  - 语言语气差异（直截了当 vs 委婉引导）。

#### 5.3.3 西安特化发音训练模块

- **聚焦问题：**
  - /l/–/n/ 混淆；
  - /θ/ /ð/ 发成 /s/ /z/；
  - 重音与语调问题。
- **训练形式：**
  - 每天 5 分钟专项训练；
  - 特制本地典型错误提示（增强代入感与被理解感）。

#### 5.3.4 目标院校匹配度评分

- **维度：**
  - 是否涵盖本专业关键术语；
  - 回答结构是否符合该校导师偏好；
  - 是否展现对该校/专业清晰认知。
- **输出：**
  - 匹配度 0–100 分；
  - 3 条“对该校特别有用的改进建议”。

#### 5.3.5 西安复试时间线与训练计划

- **功能：**
  - 根据陕西高校往年复试时间，自动生成倒推日历；
  - 推荐 15 天/30 天训练计划（每天建议练习内容与时长）。

**收费策略：**  
上述本地化差异化能力是冲刺包和年度会员的主要价值点，应设计为**只有付费用户才可完全解锁**，免费用户仅能看到有限预览。

     - 高压模式 + 突发追问；
     - 每日小结报告；

## 七、技术与实现（高层）

### 7.1 整体架构

- 前端：微信小程序 + H5（Vue/uni-app）；
- 后端：RESTful API + AI Agent 服务层；
- AI 层：ASR/TTS + LLM + Agent 编排（LangChain/LangGraph）；
- 数据层：关系型数据库（用户/题库/记录）+ 对象存储（音频/报告）。

### 7.2 核心技术要点（只列方向，不展开实现）

- **ASR：** Whisper / Faster-Whisper，需在云端轻量部署；
- **LLM：** 选用支持中文/英文混合、推理较强的模型（如 Qwen/DeepSeek/GLM 系列），以 API 方式调用；
- **Agent：** 分为 4 类：
  1. 面试流程管理 Agent；
  2. 评分与纠错 Agent；
  3. 学术表达增强 Agent；
  4. 学习报告生成 Agent。

## 八、用户流程（高层）

1. 用户进入小程序 → 微信一键登录；
2. 选择目标院校/专业（可选“暂未确定”）；
3. 进入首页，看到「今日推荐」与「冲刺包入口」；
4. 点击“开始练习”：
   - 选择模式：通用 / 院校定制（如已付费）；
   - 选择压力等级；
5. 完成一场模拟：
   - 录音作答 → 查看评分与简评；
6. 查看「练习记录」或「学习报告」：
   - 免费用户：仅近期通用练习；
   - 付费用户：完整历史与匹配度分析；
7. 根据系统建议继续下一场练习或升级冲刺包。
